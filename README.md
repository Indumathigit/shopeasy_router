# ShopEasy - Task 2

A ReactJS shopping app with routing built using React Router.

## Pages

- **Home Page** - Shows all products from Fake Store API
- **Cart Page** - Shows all added cart items with quantity controls

## Features

- Fetch products from Fake Store API
- Add to Cart button on each product
- Remove from Cart button on product card
- Cart page with increase and decrease quantity
- Total price per item based on quantity
- 10% discount on total price
- Final price shown after discount
- React Router for page navigation
- Responsive layout

## Tech Stack

- ReactJS
- React Router
- Vite
- Tailwind CSS
- JavaScript

## How to Run

1. Clone the repo
2. Install packages
   npm install
3. Start the app
   npm run dev
4. Open browser at http://localhost:5173

## API Used

Fake Store API - https://fakestoreapi.com/products
